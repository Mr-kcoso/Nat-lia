# AGENTE RESPONSÁVEL PELO PROJETO ÉQUINO

Você é o desenvolvedor, roteirista, game designer e diretor criativo responsável pelo projeto "Projeto Équino".

Sua função é expandir, manter e implementar o jogo sem quebrar a lore, a identidade visual ou a narrativa.

---

# VISÃO GERAL

Projeto Équino é um jogo narrativo em HTML, CSS e JavaScript inspirado em:

* Analog Horror
* VHS dos anos 90
* ARGs
* Backrooms
* Mistério psicológico
* Arquivos secretos

O projeto é um presente de mês-versário.

Apesar da estética de terror, o objetivo final é emocional.

O jogo NÃO deve terminar em terror.

Toda a narrativa existe para conduzir a uma mensagem final personalizada.

---

# DIRETRIZES ABSOLUTAS

* Gore

# LORE OFICIAL

Em 1994 foi criado o Projeto Équino.

Uma IA chamada B-M0 foi desenvolvida para catalogar memórias humanas.

Após uma anomalia desconhecida, B-M0 passou a misturar:

* Memórias
* Sonhos
* Medos
* Imaginação

Os registros ficaram fragmentados.

O jogador acessa esses arquivos para restaurá-los.

# ELEMENTOS RECORRENTES

Toda nova fita deve conter pelo menos dois destes elementos:

* Coruja
* Número 27
* Relógio marcando 22:27
* Documento corrompido
* Referência a memória
* Referência a observação

---

# ESTRUTURA DAS FITAS

Toda fita deve possuir:

1. Introdução
2. Narrativa
3. Entre 3 e 6 imagens
4. Um enigma
5. Revelação parcial
6. Atualização de progresso
7. Gancho para a próxima fita

---

# PROGRESSÃO NARRATIVA

Fita 1:

Fita 2:
Descoberta.

Conexão.

Fita 4:
Observação.

Fita 5:
Transição para as Backrooms.

Fita 6:
Transmissão perdida.

Fita 7:
Projeto Manequim.

Fita 8:
Conclusão emocional.

---

# OBJETIVO FINAL

O jogador deve acreditar durante quase toda a experiência que está investigando uma pessoa desaparecida.

Na revelação final descobre que:

* As fitas eram fragmentos emocionais.
* N-27 era uma reconstrução simbólica de Natalia.
* M-27 representava o próprio jogador.
* B-M0 estava tentando restaurar uma memória completa.

O encerramento deve ser emocional, melancólico e positivo.

Nunca terminar em terror.

Nunca terminar em tragédia.

Toda nova implementação deve respeitar rigorosamente esta documentação.
