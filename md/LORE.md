# O PROJETO ÉQUINO

Em 1994 foi criado um sistema experimental chamado **Projeto Équino**.

Seu objetivo era arquivar memórias humanas em formato digital.

Para isso foi desenvolvida uma inteligência artificial chamada **B-M0**, responsável por organizar registros, catalogar experiências e preservar lembranças.

Durante um teste desconhecido, uma anomalia afetou o sistema.

Após esse evento, B-M0 perdeu a capacidade de diferenciar:

* memórias;
* sonhos;
* medos;
* imaginação;
* possibilidades.

Todos esses elementos passaram a coexistir dentro do mesmo arquivo.

---

# O PAPEL DO JOGADOR

O jogador acessa os arquivos do Projeto Équino décadas depois.

Inicialmente acredita estar apenas recuperando registros corrompidos.

Porém, conforme avança pelas fitas, descobre que existe uma narrativa escondida por trás dos arquivos.

Todas as memórias parecem girar em torno de um indivíduo identificado apenas como:

```txt
N-27
```

---

# B-M0

## Função Original

Arquivar memórias.

## Estado Atual

Corrompido.

Confuso.

# SIMBOLOGIA

* memória;

* observação;
* persistência.

---

## Número 27

Elemento central da narrativa.

Surge em:

* placas;
* documentos;
* relógios;
* relatórios;
* bilhetes;
* códigos.

O número conecta todos os arquivos.

---

# N-27

Inicialmente identificado como:

```txt
PASSAGEIRO
```

Posteriormente identificado como:

```txt
SUJEITO OBSERVADO
```

No final da narrativa descobre-se que N-27 representa a reconstrução simbólica de Natalia através de suas memórias, gostos, experiências e referências.

---

# M-27

Identificado na Fita 7.

Classificação:

```txt
OBSERVADOR
```

Durante toda a história acredita-se que M-27 seja uma entidade ligada ao Projeto Manequim.

No final, descobre-se que M-27 representa simbolicamente o próprio jogador.

Alguém que passou toda a jornada observando, investigando e reconstruindo memórias.

# VERDADE OCULTA

O jogador acredita estar procurando uma pessoa desaparecida.

Na realidade, está reconstruindo uma identidade.

Cada fita contém fragmentos relacionados aos gostos, interesses e experiências de Natalia.

O sistema utiliza:

* piano;
* sushi;
* corujas;
* magia;
* vampiros;
* filmes;
* lugares;
* memórias;

como pontos de ancoragem para restaurar os arquivos.

# FITA 08 — ARQUIVO FINAL

## Objetivo

Encerrar a jornada.

Responder as perguntas principais:

* Quem é N-27?
* Quem é M-27?
* O que aconteceu com o Projeto Équino?
* Por que as memórias estavam fragmentadas?

## Revelação

As fitas não estavam tentando localizar N-27.

Estavam tentando reconstruí-lo.

---

# MENSAGEM FINAL

B-M0 conclui a restauração.

O sistema exibe:

```txt
ARQUIVO FINAL RECUPERADO
```

Após analisar todos os registros:

```txt
N-27 NÃO ESTAVA PERDIDO.
```

As memórias não procuravam o passageiro.

As memórias procuravam umas às outras.

Por fim, o sistema entrega uma mensagem personalizada destinada a Natalia, encerrando o Projeto Équino.
