# VISÃO GERAL

Projeto criado como presente de mês-versário.

Formato:

* Jogo HTML/CSS/JS
* Estética VHS 1994
* Analog Horror psicológico
* Mobile First
* Narrativa baseada em memórias recuperadas

---

# OBJETIVO NARRATIVO

O jogador assume o papel de uma pessoa que acessa os arquivos do Projeto Équino.

Uma IA chamada B-M0 auxilia na recuperação das memórias.

Inicialmente parece um sistema de arquivamento antigo.

Ao longo da história descobre-se que:

* As memórias estão conectadas.
* Existe uma entidade chamada N-27.
* Existe um observador chamado M-27.
* Algumas memórias foram alteradas.
* As Backrooms possuem ligação direta com os registros.

O objetivo final é reconstruir completamente a história de N-27.
